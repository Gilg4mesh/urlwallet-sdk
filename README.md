# About

## Develop
```
npm run dev # http://localhost:8080/dev.html for develop page and http://localhost:9080 for scripts
```

## Usage

```
npm install
npm run build		# for url-wallet.js and urlwallet-sdk.js
npm run build:prod	# for url-wallet.min.js and urlwallet-sdk.min.js
```
